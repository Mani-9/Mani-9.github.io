Introducción a CSS


¿Qué es CSS y para qué sirve?


¿Qué es CSS?

Es un lenguaje de marcas enfocado a definir, crear y mejorar la presentación de un documento basado en HTML.


¿Para que sirve CSS

El código CSS hace la vida más fácil al desarrollador front-end al separar las estructura de un documento HTML de su presentación. Dicho de otro modo el HTML actuaría como es esqueleto de la web, definiendo su estructura básica, y el CSS añadiría toda la capa de personalización sobre el que la web define su aspecto final.


Formas de aplicar CSS

  1. En línea:

    Consiste en indicar los estilos de un elemento HTML dentro de su etiqueta de apertura, mediante un atributo HTML.

    <html>
    <head>  
        <title>CSS integrado en las etiquetas</title>  
    </head> 
    <body> 
           <p style="color:green;">Párrafo de color verde.</p> 
    </body> 
</html>

  2. Interna:

    Consiste en crear un documento CSS muy aparte y guardarlo con la extensión.

        ul {
        list-style-type: none;
        margin: 0;
        padding: 0;
        overflow: hidden;
        background-color: #333;
        font-family: Arial, Helvetica, sans-serif;
    }
    
    li {
        float: left;
    }

  3. Externa:

    Consiste en crear en el documento HTML los estilos que deseamos aplicar

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        #footer {
            background-color:rgb(14, 14, 75);
            position: absolute;
            display: inline-block;
            bottom: auto;
            width: 100%;
            height: 40px;
            color: white;
        }
   </style>
</head>
<body>
                            
</body>
</html>

Qué es un comentario y como se escribe en CSS.


  1. Qué es un comentario?

    Los comentarios son usados para añadir notas explicatorias o prevenir que el navegador interprete partes de la hoja de estilos. Los comentarios se pueden colocar en cualquier espacio en blanco que se permita en la hoja de estilos.


  2. Como se escribe en CSS.

        SINTAXIS:
    /* Comentario */

    EJEMPLO:
    /* Comentario de una sola línea */

    /*
    Un comentario
    que se extiende
    en varias
    líneas
    */
