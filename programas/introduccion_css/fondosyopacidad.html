Fondos y opacidad


  1. Background-color,:

    Este Background especifica el color de fondo de un elemento.


    body {
  background-color: lightblue;
}order: red 25px solid;

  2. background-image:

    Este Background especifica una imagen para usar como fondo de un elemento. 


    body {
  background-image: url("paper.gif");
}

  3. Background-repeat:

    Este Background repite una imagen tanto horizontal como verticalmente.

    body {
  background-image: url("gradient_bg.png");
  background-repeat: repeat-x;
}


  4. Background-attachment:

    Este Background especifica si la imagen de fondo debe desplazarse o ser fija (no se desplazará con el resto de la página):.


    body {
  background-image: url("img_tree.png");
  background-repeat: no-repeat;
  background-position: right top;
  background-attachment: scroll;
}

  5. Background-position:

    Este Background define la posición inicial de la imagen de fondo especificada.


    body {
  background: #ffffff url("img_tree.png") no-repeat right top;
}

  6. Propiedad opacity.

    Esta propiedad define la transparencia de un elemento, esto es, en qué grado se superpone el fondo al elemento.


    div {
  width: 200px;
  height: 200px;
  background: #FF7A59;
  opacity: .5;
}
