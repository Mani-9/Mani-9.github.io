Propiedades de sombreado


Propiedades text-shadow y box-shadow.


  1. Propiedad text-shadow:

    Epermite agregar sombras al texto.


    .red-text-shadow {
    text-shadow: red 0 -2px;
}

    2. Propiedad box-shadow:

    se utiliza para aplicar una o más sombras en un elemento.


    div {
  box-shadow: 10px 10px 5px 12px lightblue;
}
