Bordes


  1. Que es un borde?

    Es una propiedad que te permite definir los bordes alrededor de los elementos HTML.


    .mi-elemento {
border: 1px solid #000;
/* Un borde sólido de 1 píxel de ancho y color negro */
}

Tipos de bordes


  1. dotted:

    Define un borde punteado


    p.dotted {border-style: dotted;
}

  2. dashed:

    Define un borde discontinuo


    p.dashed {border-style: dashed;
}

  3. solid:

    Define un borde sólido


    p.solid {border-style: solid;
}

  4. double

    Define un doble borde


    p.double {border-style: double;
}

  5. groove:

    Define un borde ranurado 3D. El efecto depende del valor del color del borde 


    p.double {border-style: groove;
}

  6. ridge:

    Define un borde estriado 3D. El efecto depende del valor del color del borde


    p.double {border-style: ridge;
}

  7. inset:

    Define un borde de inserción 3D. El efecto depende del valor del color del borde 


    p.double {border-style: inset;
}

  8. outset:

    Define un borde de inicio 3D. El efecto depende del valor del color del borde 


    p.double {border-style: outset;
}

  9. none:

    No define ningún borde


    p.double {border-style: none;
}

  10. hidden:

    Define un borde oculto


    p.double {border-style: hidden;
}

Colores CSS


  1. Nombre:

    La propiedad puede tener de uno a cuatro valores (para el borde superior, el borde derecho, el borde inferior y el borde izquierdo).


    p.one {
  border-style: solid;
  border-color: red green blue yellow;
  /* red top, green right, blue bottom and yellow left */
}

  2. HEX:

    El color del borde también se puede especificar mediante un valor hexadecimal (HEX):


    p.one {
  border-style: solid;
  border-color: #ff0000; /* red */
}

  3. RGB:

    Uso de valores RGB 


    p.one {
  border-style: solid;
  border-color: rgb(255, 0, 0); /* red */
}

  4. HSL

    Uso de valores HSL


    p.one {
  border-style: solid;
  border-color: hsl(0, 100%, 50%); /* red */
}

Ancho de borde


Esta propiedad permite establecer el grosor de los cuatro bordes de un elemento. Puede tener de uno a cuatro valores. 


element {
    border-width: thin;
    border-style: solid;
    border-color: #000;
}

Bordes redondeados.


  1. Cuatro valores :

    border-radius: 15px 50px 30px 5px; Primero el valor se aplica a la esquina superior izquierda, el segundo valor se aplica a la esquina superior derecha, El tercer valor se aplica a la esquina inferior derecha y el cuarto valor se aplica a esquina inferior izquierda: 


    #rcorners1 {
  border-radius: 15px 50px 30px 5px;
  background: #73AD21;
  padding: 20px;
  width: 200px;
  height: 150px;
}

  2. Tres valores:

    border-radius: 15px 50px 30px; Primer valor se aplica a la esquina superior izquierda, el segundo valor se aplica a la esquina superior derecha y a la parte inferior izquierda esquinas, y el tercer valor se aplica a la esquina inferior derecha:


    #rcorners2 {
  border-radius: 15px 50px 30px;
  background: #73AD21;
  padding: 20px;
  width: 200px;
  height: 150px;
}

  3. Dos valores :

    border-radius: 15px 50px; Se aplica el primer valor a las esquinas superior izquierda e inferior derecha, y el segundo valor se aplica a la esquina superior derecha y esquinas inferiores izquierdas:


    #rcorners3 {
  border-radius: 15px 50px;
  background: #73AD21;
  padding: 20px;
  width: 200px;
  height: 150px;
}

  4. Un valor

    border-radius: 15px; El valor se aplica a todos los cuatro esquinas, que se redondean igualmente:


    #rcorners4 {
  border-radius: 15px;
  background: #73AD21;
  padding: 20px;
  width: 200px;
  height: 150px;
}
