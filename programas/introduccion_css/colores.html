Colores


    Como se especifican los colores en CSS:


  1. Usando una palabra clave:


    Es como una variable que contiene el valor actual de la propiedad de color de un elemento. Esta palabra clave puede ser útil si desea que un color específico sea consistente en un elemento o una página.


    div {
  color: blue;
  border: 10px solid currentcolor;
}

  2. Usando el sistema RGB cubic-coordinate:


    Consiste en definir un color indicando la cantidad de color rojo, verde y azul que se debe mezclar para obtener ese color. 


    h2 {
  background-color: rgb(150,100,255);
}
p {
    background-color: rgb(150,100,255,.5);
  }

  3. Usando el sistema HSL cylindrical-coordinate:


    Se basa en la rueda de colores RGB. Cada color tiene un ángulo y un valor porcentual para los valores de saturación y luminosidad.


    .element {
    background-color: hsl(196, 73%, 62%);
}

  4. Colores gradiantes:


    Se pueden usar en cualquier lugar donde usaría <image>, como en los fondos. Debido a que los degradados se generan dinámicamente, pueden anular la necesidad de los archivos de imagen de trama que tradicionalmente se usaban para lograr efectos similares.


    Ejemplos tipos de gradiantes:


    Gradiante lineal:


    #grad {
  background-image: linear-gradient(red, yellow);
}

    Gradiante radial:


    #grad {
  background-image: radial-gradient(red, yellow, green);
}

    Gradiante conica:


    #grad {
  background-image: conic-gradient(red, yellow, green);
}
